# HandX

> *Hand of Developer* - Inspired by "Hand of the King" from Game of Thrones

## What is HandX?

HandX is your intelligent development agent that works on your computer or server while you command it from anywhere - even your mobile phone.

Like the Hand of the King serves the throne, HandX serves the developer.

## Key Features

- **Remote Agent**: An intelligent agent that runs on your desktop or server
- **Mobile Control**: Command and monitor your development tasks from your phone
- **Always Working**: Your code assistant that works even when you're away from your desk

## Use Cases

- Check build status while commuting
- Trigger deployments from your phone
- Monitor CI/CD pipelines on the go
- Execute development tasks remotely
